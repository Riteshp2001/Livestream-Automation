# Stream Sound Attributions

This repo now carries a migrated stream sound library under `assets/stream_sounds/` so generated livestreams do not depend on `auraninja/` at runtime.

## Ambient, Rain, Nature, Places, And Thing Layers

- Source note inherited from Auraninja: ambient sounds were documented there as sourced from Pixabay.
- These files are tracked in `stream_sound_catalog.json` with `license_source = "Pixabay Content License"`.
- Treat those entries as internal carryover metadata from the source project.

## Binaural Layers

- Source note inherited from Auraninja: binaural assets were documented there as licensed from Binauro.
- These files are tracked in `stream_sound_catalog.json` with `license_source = "Binauro"`.

## Noise Layers

- The bundled noise files were migrated as internal carryover from Auraninja.
- They are tracked in `stream_sound_catalog.json` as inherited internal assets and should be re-verified before any external redistribution.
